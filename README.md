## 📌 Featured Projects

### 🎵 [Music Room](https://github.com/ennaciririda/music-room)
Real-time collaborative music app with **Track Vote** and **Playlist Editor** services. Built with Node/Express, TypeScript, Socket.IO, and Prisma/PostgreSQL — with atomic operations for vote concurrency and versioned-state conflict resolution for collaborative playlist reordering.

### 🏓 [FT_TRANSCENDENCE (Pongiverse)](https://github.com/ennaciririda/FT_TRANSCENDENCE)
Full-stack multiplayer Pong platform with Django, PostgreSQL, WebSockets, JWT + 2FA auth, and Grafana/Prometheus monitoring.

### 🎬 [Hypertube](https://github.com/ennaciririda/hypertube)
Torrent-based movie streaming platform — contributed the frontend using Next.js, Redux Toolkit, shadcn/ui, and Tailwind CSS.

### ☸️ [Inception-of-Things](https://github.com/ennaciririda/Inception-of-Things)
Multi-part Kubernetes infrastructure project using Vagrant, K3s, K3d, and Argo CD (GitOps), with GitLab integration as a bonus.

### 📱 [Swifty Companion](https://github.com/ennaciririda/swifty-companion)
React Native (Expo) mobile app consuming the 42 API via OAuth2 client-credentials flow.
